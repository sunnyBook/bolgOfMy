CSS 2.0 (一)

发表于 2017-04-23 | 分类于 CSS | | 阅读次数

什么是CSS？

css全称为cascading style sheet层叠样式表，他的主要作用就是为html标签添加各种各样的样
式。

引入方式

1.行间样式：直接在html标签里面写style标签

===========================================
2.页面级css：在head标签中添加一个style标签

  • 这里的type属性可以不写，如果写就一定要写对。

===========================================
3.外部css文件：一般都使用这种形式的引入方式，在外面创建一个.css的文件，然后在html页面 中引入即可。

href属性值是css文件的地址，最好是相对地址
link加载不会阻塞html加载，html和css是异步加载

===========================================
4.import引用（已经弃用）
在head标签中写一个style标签，在第一行写上@import url(); url中写css文件地址，可以加引号也可以不加。

弃用原因：
        1.必须写在第一行，若多个则一起写在最前面。
        2.IE6环境下，最多只能使用31次。
        3.import引入方式和html是同步加载方式。

选择器

  1. id选择器： 给元素添加一个id属性，一个元素只能有一个id，一个id也只能给一个元素。然后在css文件中，通过 #id{}，就可以选择我们要添加id的那个元素了。
  1. class选择器： 给元素添加class属性，一个元素可以有多个class属性，一个class也可以赋给 多个元素。在css文件中，通过.class{}方式选择。
  1. 标签选择器： div{} 这样是要是div标签就被选择出来了
  1. 通配符选择器： *{}所有的标签都会被选择出来

在单独的样式后面加上 !important后，这个样式就被赋最高的优先级，
在没有特殊情况下，后面怎么样添加样式都不会覆盖或者修改这个样式
例如：background-color : red ! important;

  1. 属性选择器： [id] {} 这样有id属性的标签就会被选择出来了。

  2. 父子选择器：也叫派生选择器 div p {} 选择div下面的p标签,这是p的权重值是加和的结果

  3. 直接子元素选择器：div > strong{}

  4. 并列选择器：<div class='select'></div>使用div.select{}选出类名是select的div

  5. 分组选择器：<div></div><p></p>这样div,p{}可以把他们都选择出来

    测试发现选择器的优先级如下：

    ! important > 行间样式 > id > (class、伪类、属性) > (标签、伪元素) > 通配符

文字样式

font-size: 设置文字大小，文字默认16px，这条属性其实设置的是文字的高

font-weight: 设置文字粗细程度,这条属性常用值有, light, normal, bold, bolder,其次也可以写100-900的数字。

font-family: 设置文字的样式,默认的是arial字体

font-style: 这个属性设置文字是否是斜体，italic就是斜体的意思。

color: 设置文字颜色，文字颜色有三种表达方式：

  • 英文单词：red，orange…..
  • 三位十六进制：每个值都是00-ff
  • rgb(xx,xx,xx)：里面是0-255的十进制数

text-indent: 设置文字首行缩进，值有两种px或em

  • 一种相对长度单位，因为每个像素点大小不同
  • 1em = 1 * font-size

border：这是个复合属性，由border-width,border-style,border-color三个复合成的 solid (实线) dotted (短虚线) dashed (长条虚线)

text-align: 设置文字的位置属性 center居中 left左对齐 right右对齐

line-height: 设置一行文字所占高度的属性，默认和字体大小一样, 如果想让单行文字在容器内上下居中，只需让height = line-height

text-decoration: 文字装饰的意思，可以设置文字是否有下划线，上划线，中划线，值分别为 underline, overline, line-through

cursor: cursor:pointer; 当鼠标移入这个元素，变成小手

伪类

hover伪类：可以让我们设置当鼠标移入这个元素的时候会改变元素的样式
        a:hover {
            font-size : 20px;
            color : #424242;
        }
